# aoc-2025
Advent of Code 2025

## Setup

This project uses [Bun](https://bun.sh/) as the JavaScript runtime.

## Running Solutions

To run a specific day's solution:

```bash
npm run day -- <day>
```

For example, to run day 1:

```bash
npm run day -- 1
```

## Project Structure

├── day2/
